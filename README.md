# Estia

A personal OS for ambitious people — goal planning, habit tracking, time blocking, brain dump & AI-powered daily planning.

## Tech Stack
- **Frontend** — Next.js 14, Tailwind CSS, TypeScript
- **Backend** — Next.js API Routes, MongoDB, Mongoose
- **Auth** — JWT, bcrypt
- **AI** — Groq (Llama 3)

## Features
- [ ] Goal planning with milestone tracking
- [ ] Habit tracking with forgiving streak logic
- [ ] Time blocking with drag-and-drop calendar
- [ ] Brain dump with AI sorting
- [ ] Vision board
- [ ] AI assistant — plan your day, panic mode, weekly review
